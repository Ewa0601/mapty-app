## 🗺️ Mapty App

A workout tracking application that allows users to log running and cycling activities on an interactive map. The app uses geolocation to display the user's position and enables storing workouts with detailed metrics.

---

## 🚀 Features

- Geolocation to detect user’s current position
- Interactive map with workout markers
- Logging running and cycling workouts
- Displaying workout details (distance, duration, cadence/elevation)
- Sidebar with workout history
- Clicking a workout moves the map to its location
- Data persistence using local storage

---

## 🛠️ Technologies

- HTML
- CSS
- JavaScript (ES6+)
- Leaflet.js (interactive maps)
- Geolocation API
- Local Storage API

---

## 🧠 Key Concepts

This project focuses on:

- Object-Oriented Programming (OOP)
- Working with external libraries (Leaflet)
- Geolocation and browser APIs
- Event handling and DOM manipulation
- Data persistence in the browser

---

## 🎮 How It Works

1. Allow the app to access your location
2. Click anywhere on the map to add a workout
3. Enter workout details (type, distance, duration, etc.)
4. View workouts on the map and in the sidebar
5. Click a workout to navigate to its position

---

## 📚 Purpose

This project was created as a learning exercise to practice advanced JavaScript concepts such as OOP, working with APIs, and building interactive applications with real-world functionality.
